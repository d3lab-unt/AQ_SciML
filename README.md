# Benchmarking Air Quality Data using Scientific Machine Learning Methods

A reproducible **Scientific Machine Learning (SciML)** benchmarking suite for **multi-horizon AQI forecasting** using **physics-guided neural networks**.
This project evaluates **MLP+Physics** and **LSTM+Physics** models on **PM₂.₅** and **Ozone** AQI datasets (2022–2024) with a **λ-sweep** to balance **data loss** and **physics consistency loss**.
